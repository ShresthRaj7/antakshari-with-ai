🎵 How to Play Antakshari

Rules:

  • 🎯 Take turns naming songs with the AI
  • 🔤 Each song must start with the last letter of the previous song
  • ⏰ You have 30 seconds per turn
  • 🚫 No repeating songs
  • 🏆 First to run out of songs loses!

Example:

Song: "Hello" → Next song starts with "O" → "Over the Rainbow"
